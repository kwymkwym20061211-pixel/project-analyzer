# 📝 ProjectAnalyzer: Project Statistics & Structure Tool

ProjectAnalyzer is a specialized tool designed to scan a target project root, analyze its source code distribution, and provide a comprehensive statistical breakdown directly to your console.

## 🌐 Overview (概要)

This tool automates the tedious task of manual code counting and structure mapping. It provides developers with an immediate "bird's-eye view" of their codebase's health, complexity, and composition.
